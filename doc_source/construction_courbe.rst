Généralités
===========

Ce module définit les classes suivantes pour décrire les propriétés de la courbe étudiée :
    - :py:class:`construction_courbe.Symetrie`
    - :py:class:`construction_courbe.Tangente`
    - :py:class:`construction_courbe.PointSingulier`
    - :py:class:`construction_courbe.Asymptote`
    - :py:class:`construction_courbe.AsymptoteHorizontale`
    - :py:class:`construction_courbe.AsymptoteVerticale`

La classe principale, qui utilise toutes ces informations est la classe :py:class:`construction_courbe.PasAPas`.
