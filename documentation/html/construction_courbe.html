Le module principal¶

Généralités¶

Ce module définit les classes suivantes pour décrire les propriétés de la courbe étudiée :

  • construction_courbe.Symetrie

  • construction_courbe.Tangente

  • construction_courbe.PointSingulier

  • construction_courbe.Asymptote

  • construction_courbe.AsymptoteHorizontale

  • construction_courbe.AsymptoteVerticale

La classe principale, qui utilise toutes ces informations est la classe construction_courbe.PasAPas.

Les classes définies¶

Module de base pour la construction de courbes paramétrée animée par manim.

class construction_courbe.Asymptote(equation, couleur='#FFFFFF')[source]¶
Paramètres:

  • equation (tuple ou list) – est un triplet (a, b, c) tel que l’équation de l’asymptote est ax + by + c = 0, XOR une paire (a, b) telle que l’équation est y = ax+b

  • couleur (str) – est optionnel et permet de choisir la couleur de tracé

Les nombres passés seront utilisés pour l’affichage et le calcul. Utiliser fractions.Fraction si nécessaire. Dans certains cas, les nombres seront divisés les uns par les autres.

L’équation sera affichée sous la forme y=ax+b ou x=c

class construction_courbe.PasAPas(*args, **kwargs)[source]¶

Classe de base à sous-classer pour obtenir une vidéo.

En plus des méthodes, le comportement de cette classe est donné par ses attributs de classe.

Gestion du repère et du tracé¶

X_RANGE: tuple de longueur 2 ou 3 = (-1.2, 1.2, 1)¶

Extrémités de l’axe des abscisses, plus un pas optionnel.

Y_RANGE: tuple de longueur 2 ou 3 = (-1.2, 1.2, 1)¶

Extrémités de l’axe des ordonnées, plus un pas optionnel.

P_RANGE: tuple de longueur 2, ou liste de tels tuples = None¶

Bornes de l’intervalle de paramètres à utiliser pour le tracé. Voir les discontinuités pour l’utilisation d’une liste.

DISCONTINUITES: list = None¶

Liste des paramètres où la courbe n’est pas continue. A utiliser en lien avec DT

DT: float = 1e-1¶
tolérance à gauche et à droite aux points de discontinuité.

Pour gérer les discontinuité nous avons 2 choix : - Donner plusieurs intervalles de paramètres - Utiliser DISCONTINUITES et DT comme le fait manim de base. Ce rendu peut être non symétrique et l’animation peut varier en vitesse si les intervalles de tracé ne sont pas de même longueur.

Les étapes de la construction¶

Explication d’une construction de courbe¶

L’utilisation des attributs suivants est implémenté dans la méthode quelques_points().

PARAMS: list = []¶

Une liste de paramètres où afficher les points de la courbe, ainsi que les points sur les axes correspondants.

TEXTES_POINTS: list = []¶

Une liste de même longueur que :PARAMS qui donne les objets manim texte à afficher comme explication à droite du repère.

LEGENDES: list = []¶

Une liste de même longueur que :PARAMS qui contient des dictionnaires dont les clées sont « x », « y », « M » et donne les positions relatives des légendes pour les trois points placés pour chaque paramètre.

Les propriétés de la courbe¶

SYMETRIES: list = []¶

La liste des Symetrie présentées par la courbe, éventuellement vide.

POINTS_REGULIERS: list = []¶

La liste de tuple (str, Tangente) où on veut l’étude des tangentes, éventuellement vide. Le premier élément est la chaîne représentant le paramètre.

POINTS_SINGULIERS: list = []¶

La liste des PointSingulier étudiés, éventuellement vide

ASYMPTOTES: list = []¶

La liste des ASYMPTOTES que possède la courbe, éventuellement vide.

Gestion de la vidéo¶

INTERVALLE_LEGENDE: float = 1.2¶

Temps en seconde à attendre avant de poursuivre l’affichage.

EPISODES: bool = True¶

Annonce ou non des parties de l’étude sous forme d’épisodes numérotés.

SKIP: bool = False¶

Ne génére pas la partie étude de courbe, mais seulement la partie tracé.

class construction_courbe.Symetrie(transforme, sym_M, texte, parametres, presentation, conclusion, intervalles, p_range, couleur='#FFFFFF')[source]¶

Les instances de cette classe représentent les éventuelles symétrie de la courbe étudiée.

Paramètres:

  • transforme – est une fonction qui prend un paramètre t et renvoie la valeur du paramètre du point symétrique.

  • sym_M – est une fonction qui prend un np.array de taille 3 et renvoie son symétrique sous forme de np.array

  • texte (str) – est le texte à afficher comme paramètre pour le symétrique de M(t). Peut contenir du Latex

  • parametres – est une liste ou un tuple de deux paramètres (nombres) à utiliser comme exemple pour afficher deux M(t) ainsi que leurs symétriques.

  • presentation – est un Text (ou Tex) de description du point symétrique. Par exemple « M(-t) = (-x(t), y(t)) »

  • conclusion – est un Text de conclusion sur la symétrie. « M(-t) est le symétrique de M(t) par rapport à (Oy) »

  • intervalles – est une liste de 3 str contenant les intervalles : complet, pour M(t), pour son symétrique. Le premier intervalle n’est pas utilisé pour l’instant.

  • p_range – donne les bornes (nombres) de l’intervalle réduit pour le futur tracé. Séquence de longueur 2 qui prendra le pas sur les paramètres donnés à PasAPas.

  • couleur – (optionnel) est chaîne de couleur à utiliser pour le tracé du morceau de courbe symétrique.
